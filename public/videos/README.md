# Video Assets

## Hero Video

Place your looping massage video here as `massage-hero.mp4`.

### Requirements:
- Format: MP4
- Resolution: 1920x1080 (16:9 aspect ratio)
- Duration: 10-30 seconds (will loop)
- File size: Optimize for web (recommended: < 5MB)
- Content: Close-up massage footage, suitable for black-and-white filter

### Notes:
- The video will automatically have a grayscale filter applied via CSS
- The video should loop seamlessly
- Ensure the video is muted (required for autoplay in browsers)
